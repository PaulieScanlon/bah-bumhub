#### [Dropdown](#dropdown)

<Grid sx={{ gap: 4}}>

Dropdown is used in the main navigation and can be used to navigate to internal routes or external urls. The trigger
render prop can compose any React component.

```javascript
<Dropdown
  trigger={<MenuButton aria-label="Menu" />}
  items={[
    {
      name: 'BumHub',
      to: '/',
    },
    {
      name: 'Bum Search',
      to: '/search',
    },
    {
      name: 'paulie.dev',
      url: 'https://paulie.dev/',
    },
  ]}
/>
```
